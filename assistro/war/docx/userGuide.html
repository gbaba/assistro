  1. Copy Our Code Into Your Web Site

Great news! You do not need a web expert to get the Assistro code on your site. It just entails copying and pasting of the chat box code from the install tab in your dashboard on Assistro.com. After logging in to your account, you can generate your code here to copy and paste onto the desired page of your site where the chat box would appear.



Just follow the simple step-by-step instructions for manual installation:



  • Copy our JavaScript code

  • Open your webpage in a text editor. A text editor is a type of program which is used for editing plain text files. (i.e., Notepad, TextEdit, Mousepad, Leafpad etc.)

  • Paste the Assistro code anywhere in the webpage’s HTML source. However, we strongly recommend that you place the code in the footer of your page just before the </body> tag

  • Save your webpage

  • Upload it to your webserver

  • Congratulations! You’re done copying the Assistro code into your website. No sweat, right?



For other CMS platforms:



To add Assistro to a Wordpress site:

  • Open your Wordpress installation

  • Go to your theme

  • Select edit footer.php

  • Copy the generated assistro code and paste it above the last </body> tag

  • Click save

Point to remember: This is only applicable for WordPress sites that you run yourself (your own domain) and not Wordpress.com.blogs.

To install Assistro on your Blogger page:

  • Click Add Widget

  • Save your layout

To add Assistro to your Joomla:

  • Log in to your Joomla admin area

  • Click Extensions from the drop down list and choose Template Manager

  • Click Templates tab near the Styles tab which can be found just below Template Manager: Templates

  • Click your existing theme

  • Then click “Edit main page template”

  • Scroll down to editor panel and paste your Assistro code above the </body> tag.

  • Click “Edit error page template” just below the “Edit main page template”

  • Paste your Javascript into the foot of the panel (same with what you did on the Main page template)

  • Save

To install Assistro on a Magento website:

  • Log into your Admin account

  • Click “System” which can be found on the top bar

  • Click Configuration which can be found on the bottommost of the drop down list

  • Look at the leftside area of the page and select “Design” under Configuration

  • In the Design Editor, go down to the Footer section and find the Miscellaneous HTML box.

  • Copy and paste your Assistro JavaScript code inside the box

  • Save Configuration

  • Go to Magento Store Front to check if the chat box loads in the lower left corner of the page

To install Assistro on your Wix account:

  • Log in to your Wix account

  • Click “Edit”

  • Select “Add” in order to add your Assistro code

  • Click “Widgets”

  • Select “HTML”

  • Choose “Blank HTML”

  • On the right side of the page, click “Settings”

  • Copy and paste the Assistro code

  • Make the HTML box about twice as big as it is

  • Click “Publish”

  • Check if the chat box is loaded properly. Try making the HTML box bigger in the editor if it is cut off when you expand it.

  1. Download a 3rd Party Chat Client/Configure your Instant Messenger

The moment our widget appears on your website, it’s time to connect your chat client. Downloading a 3rd party instant message client to begin receiving chats from your visitors. We recommend the following



  • WEB BASED - Meebo or Google Talk

Setting up Meebo

  1. Go to http://www.meebo.com/messenger

  2. Click the lightbulb icon

  3. Log in using your Assistro username and password

  4. Chat with your visitors from different web browsers



Setting up Google Talk:

  1. Link your Google Talk account to us

  2. Follow the instructions to log in to your Google account. Grant Assistro access.

  3. Log in to your Gmail account with your original username and password

  4. Your status should be “Available”

  5. Accept all buddy requests for you need all the contacts to make it Assistro function

  6. Now you can chat with your visitors



  • MAC - Adium or iChat

Setting up Adium:

  1. Launch Adium

  2. Choose “Preferences”

  3. Click “Accounts”

  4. Click “+” to add your Assistro Jabber account. You’ll see XMPP. That’s also Jabber.

  5. Log in using your Assistro username and password

  6. Click OK and not “Register New Account”

  7. Click “Available” from the Status drop down list to make you appear online

Setting up iChat

  1. Launch iChat

  2. Choose “Preferences”

  3. Click “Accounts”

  4. Click “+” to add a new account

  5. Select Jabber as your account type

  6. Plug in your Assistro username and password

  7. Set your status as “Available” so everyone can see that you’re online



  • WINDOWS - Pidgin

Setting up Pidgin:

  1. Launch Pidgin

  2. Click “Add”

  3. Choose “XMPP” aka Jabber

  4. Fill in the Screen name with your Assitro username; domain with Assistro.com; and password with your Assistro password

  5. You can tick the box “Remember password” if desired

  6. Click Save

  7. To enable your Assistro account, just go to “Accounts” menu option under “Enable account” option and then select Assistro from the drop down list.



  • MOBILE - Meebo (iPhone, Android and Blackberry) and BeeJive

Setting up Meebo for iPhone

  1. After creating your Meebo account in your iPhone, choose Jabber which can be found on the bottom part

  2. Enter your Assistro username and password

  3. There will be option to add other networks but you can just click “Done”

  4. Change your status to “Available”



Setting up Beejive (for iPhone, Android and Blackberry)

  1. After downloading Beejive, launch it

  2. Open the “Accounts” tab and click “+” to add a new account

  3. Choose Jabber

  4. Enter your Assistro username and password

  5. Click Save

  6. Make your status “Available”

  7. Chat with your Buddy list
